# proyecto-modulo-3
Entrega de proyecto modulo 3

En archivo datos.py se agregaron las credenciales para poder ingresar al modulo de inscipcion,
los cursos que estàn disponibles para inscripciones, se agrega lista para inscripciones y se agrega un set para mantener los nombre de los alumnos, para iterar con ellos.

En archivo funciones:
- use las funciones bàsicas como input() para capturar y print() formateando con f-strings para que todo se vea màs ordenado en la pantalla de la terminal.
- Añadì una validaciòn con .isdigit() para evitar que el programa se rompa ("crashee") si alguien mete letras en vez de los nùmeros del menu o de los còdigos de curso.

En archivo main:
El diccionario me ayudo a tener la clave del curso ligada a sus datos (nombre y cupos). El conjunto (set) es bueno porque si inscribo a un alumno en dos cursos distintos, al final me cuenta a la persona una sola vez.
